# downloadapp
How to build and run:
- On Win: copy all necessary dll file from "path_of_qt/mingw32/bin" to "path_of_source"
  1. run qmake
  2. run mingw32-make
- On Unix: install qmake if not exist
  1. run qmake
  2. run make
